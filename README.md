# Personal Website

> **NOTE: This is website was built with [Geist UI](https://github.com/geist-org/geist-ui/).**

<br/>

## Quick Start

1. run `yarn` or `npm i` to install it.

2. update seeds and metadata:

```bash
yarn docs:collect

yarn docs:seeds
```

3. development mode:

```bash
yarn dev
```
