## eth-utils.com

[eth-utils.com](https://eth-utils.com) is a collection of common utilities for working with Ethereum codebases.
I found myself googling online converters and hashing tools way too often so I decided to put these tools together in a single place.

It currently supports:

✅ Unit conversion (eg. wei, gwei)

✅ Checksum

✅ Hexstring conversion

✅ Keccak-256

You can find the source code [here](https://github.com/albertocevallos/ethutils). Feel free to create an issue if you'd like to see any new feature.
